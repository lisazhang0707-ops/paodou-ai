# 微信公众号 MCP 使用手册

**用自然语言，轻松管理你的公众号**

---

## 什么是公众号 MCP？

公众号 MCP（Model Context Protocol）让你可以用**自然语言**直接操作微信公众号，无需登录微信公众平台后台，无需写代码。

> 💡 **一句话理解**：就像你对助理说话一样，直接告诉 AI "帮我发一篇文章"，AI 会自动调用公众号接口完成操作。

### 核心优势

| 优势 | 说明 |
|---|---|
| 🚀 零代码操作 | 不需要懂 API，不需要写代码，用中文对话即可完成所有操作 |
| 🔄 批量处理 | 一次性处理多个任务，比如批量回复消息、批量获取用户数据 |
| 📊 数据分析 | 自动拉取并分析公众号数据，生成可视化报告 |
| 🤖 AI 辅助创作 | AI 帮你写文章、优化标题、生成摘要，一键发布 |

## 连接与配置

### 第一步：获取公众号凭证

1. 登录[微信公众平台](https://mp.weixin.qq.com)
2. 进入**设置与开发** → **基本配置**
3. 点击**开发者ID(AppID)** 和**开发者密码(AppSecret)** 进行获取/重置
4. 记录下 AppID 和 AppSecret（注意保密！）
5. 在 **IP 白名单**中添加服务器 IP 地址

### 第二步：在 WorkBuddy 中配置

```
💬 对 AI 说：
帮我配置微信公众号，AppID 是 xxx，AppSecret 是 xxx
```

> ✅ 配置成功后，Access Token 会自动管理，无需手动维护

### IP 白名单说明

> ⚠️ **重要提示**：微信公众号 API 要求调用服务器的 IP 必须在白名单中，否则会报错：`invalid ip ... not in whitelist`
>
> 解决方法：在公众平台后台 → 基本配置 → IP白名单，添加当前服务器 IP。

## 草稿箱管理

草稿箱是公众号文章发布前的"暂存区"，所有文章需要先存为草稿，再发布。

### 支持的操作

| 操作 | 说明 | 示例指令 |
|---|---|---|
| 📝 创建草稿 | 新建一篇文章草稿 | "帮我创建一篇关于 AI 的文章草稿" |
| 📋 查看列表 | 获取草稿箱文章列表 | "查看草稿箱里有哪些文章" |
| 🔍 查看详情 | 获取某篇草稿的详细内容 | "查看草稿 ID 为 xxx 的文章内容" |
| ✏️ 更新草稿 | 修改已有草稿的内容 | "把刚才那篇草稿的标题改成 xxx" |
| 🗑️ 删除草稿 | 从草稿箱删除文章 | "删除草稿箱里标题为 xxx 的文章" |
| 🔢 统计数量 | 查看草稿箱文章总数 | "草稿箱里一共有多少篇文章" |

### 创建草稿示例

```
💬 对 AI 说：
帮我创建一篇公众号文章草稿：
标题：2025年AI趋势展望
作者：Lisa
内容：（让AI自动生成关于2025年AI发展趋势的文章）
摘要：一文看懂2025年AI领域的五大趋势
```

> 📌 **小贴士**：创建草稿时需要先上传封面图片获取 thumbMediaId，可以让 AI 帮你先上传图片再创建草稿。

## 文章发布

将草稿箱中的文章正式发布到公众号，粉丝可以立即看到。

### 发布流程

1. 选择要发布的草稿（通过标题或 Media ID 指定）
2. AI 调用发布接口提交发布任务
3. 微信后台审核（通常几分钟内完成）
4. 发布成功后，粉丝可以在公众号看到文章

### 支持的操作

| 操作 | 说明 | 示例指令 |
|---|---|---|
| 🚀 提交发布 | 将草稿发布为正式文章 | "把草稿箱里关于AI的那篇发布出去" |
| 🔍 查询状态 | 查看发布任务的状态 | "刚才的发布任务状态怎么样了" |
| 📋 发布列表 | 查看历史发布记录 | "查看最近发布的文章列表" |
| 🗑️ 删除发布 | 删除已发布的文章 | "删除发布ID为xxx的发布记录" |

> ⚠️ **注意事项**：
> - 发布后不可撤回，请确认内容无误再发布
> - 发布后需要微信审核，通常 2-5 分钟完成
> - 一天只能群发 1 次（订阅号）或 4 次（服务号）

## 自定义菜单管理

自定义菜单是公众号底部的导航栏，用户可以点击菜单进入不同页面。

### 菜单结构

公众号底部最多支持 3 个一级菜单，每个一级菜单可包含子菜单或直接跳转链接。

### 支持的操作

| 操作 | 说明 | 示例指令 |
|---|---|---|
| 🏗️ 创建菜单 | 设置公众号底部菜单 | "帮我设置公众号菜单，第一个是首页，第二个是文章列表..." |
| 👁️ 查看菜单 | 获取当前菜单配置 | "查看当前公众号的菜单配置" |
| 🗑️ 删除菜单 | 清空自定义菜单 | "删除公众号的自定义菜单" |
| 🎯 个性化菜单 | 针对不同用户显示不同菜单 | "为男性用户设置不同的菜单" |

## 素材管理

公众号文章需要图片、视频等素材，MCP 支持上传和管理这些素材。

### 素材类型

| 类型 | 说明 |
|---|---|
| 🖼️ 图片 (image) | 文章配图、封面图等，支持 JPG/PNG/GIF |
| 🎤 语音 (voice) | 语音消息、音频内容，支持 AMR/MP3/WMA |
| 🎬 视频 (video) | 视频内容，需要填写标题和描述 |
| 📎 缩略图 (thumb) | 用于视频封面等场景的缩略图 |

### 上传素材示例

```
💬 对 AI 说：
帮我把 C:/Users/Lisaz/Downloads/cover.jpg 上传到公众号素材库，用作文章封面
```

```
💬 对 AI 说：
上传这个图片作为永久素材，并返回 media_id 供我创建草稿时使用
```

## 用户管理

管理公众号的粉丝用户，获取用户数据，设置用户标签等。

### 支持的操作

| 操作 | 说明 | 示例指令 |
|---|---|---|
| 📋 用户列表 | 获取所有关注用户的 OpenID 列表 | "获取公众号的关注用户列表" |
| 👤 用户信息 | 获取指定用户的详细信息 | "查看 OpenID 为 xxx 的用户信息" |
| 👥 批量查询 | 一次获取多个用户的信息 | "批量获取最近 50 个粉丝的信息" |
| 📝 设置备注 | 为用户设置备注名（最多50字） | "把这个用户备注设置为 VIP客户-张三" |
| 📊 用户增减数据 | 查看指定日期的用户增减情况 | "查看最近 7 天的用户增减数据" |
| 📈 累计用户数据 | 查看指定日期的累计用户数 | "查看昨天的累计关注用户数" |

> 📌 **关于 OpenID**：每个关注公众号的用户都有一个唯一的 OpenID，通过 OpenID 可以获取用户的昵称、头像、性别、地区等信息。但无法获取用户的微信账号等隐私信息。

## 常见问题排查

### 问题一：IP 不在白名单

> ⚠️ **错误信息**：`invalid ip 39.144.122.77 ipv6 ::ffff:39.144.122.77, not in whitelist`

**解决方法：**
1. 登录微信公众平台 → 设置与开发 → 基本配置
2. 找到"IP白名单" → 点击修改
3. 将报错信息中的 IP 地址（如 39.144.122.77）添加到白名单
4. 保存后等待 5-10 分钟生效

### 问题二：Access Token 过期

> 💡 **说明**：Access Token 有效期为 2 小时，MCP 会自动刷新，通常不需要手动处理。如果报错 token 无效，可以尝试重新配置公众号凭证。

### 问题三：发布失败

可能原因：
- 文章内容包含敏感词 → 修改内容后重试
- 一天内发布次数超限 → 等待次日或升级服务号
- 图片/素材未正确上传 → 检查封面图 media_id 是否有效
- 草稿格式不正确 → 检查标题、内容是否填写完整

### 问题四：菜单设置不生效

> 💡 **说明**：菜单设置后需要 5-10 分钟才能在用户端生效。如果长时间不生效，可以尝试：
> 1. 取消关注公众号，再重新关注
> 2. 在公众号对话框发送任意消息，触发菜单刷新
